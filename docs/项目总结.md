# 逐光自习室管理系统项目总结

## 项目概述

逐光自习室管理系统是一个基于Spring Boot和Vue 3的前后端分离的Web应用系统，旨在为自习室提供高效的管理和预约服务。系统实现了完整的用户管理、预约管理、支付集成、积分系统、会员系统、学习统计等核心功能。

## 技术架构

### 后端技术栈
- **框架**：Spring Boot 2.7.14
- **ORM**：MyBatis 2.3.1
- **数据库**：MySQL 8.0
- **构建工具**：Maven
- **认证方式**：JWT Token
- **支付集成**：支付宝沙箱环境
- **定时任务**：Spring Scheduled

### 前端技术栈
- **框架**：Vue 3.3.4
- **UI组件库**：Element Plus 2.4.2
- **路由**：Vue Router 4.2.5
- **状态管理**：Pinia 2.1.7
- **HTTP客户端**：Axios 1.6.0
- **构建工具**：Vite 4.5.0

## 核心特性

1. **前后端分离**：采用RESTful API进行数据交互
2. **JWT认证**：使用JWT Token实现用户身份认证
3. **角色权限**：普通用户和管理员角色区分，权限控制完善
4. **状态管理**：完整的业务状态流转（预约、租赁、支付等）
5. **数据验证**：前后端双重数据验证
6. **支付集成**：支付宝沙箱支付完整集成（异步回调、同步回调）
7. **积分系统**：签到、兑换、统计完整的积分体系
8. **会员系统**：VIP/SVIP会员购买和管理
9. **统计功能**：多维度数据统计分析
10. **座位管理**：精细化的座位状态管理和使用情况追踪
11. **时间轴可视化**：直观展示座位的使用情况时间轴
12. **隐私保护**：普通用户查看时间轴时不显示具体用户信息
13. **入场凭证**：支持预约订单和长期租赁订单的入场凭证生成
14. **留言板功能**：完整的留言、评论、点赞互动系统
15. **通知系统**：实时通知推送和未读提醒，支持管理员向用户发送通知，管理员通知加强显示
16. **学习统计**：学习时长统计、排行榜、图表可视化
17. **定时任务**：自动取消超时订单、积分发放、预约提醒
18. **管理员呼叫**：用户呼叫管理员功能（管理员页面不显示呼叫按钮）
19. **分页查询**：所有列表查询支持分页
20. **搜索功能**：支持多条件搜索和模糊搜索（用户管理支持用户名和ID模糊搜索）
21. **价格配置系统**：动态价格配置，支持管理员灵活调整所有价格和折扣参数
22. **实时刷新机制**：所有通知相关功能支持自动刷新（5秒轮询），无需手动刷新页面
23. **座位管理优化**：大区列表展示，详情页面管理，提升管理效率
24. **长期租赁优化**：与预约流程一致，选择日期范围后系统自动筛选可用座位
25. **菜单高亮提示**：呼叫记录菜单有未处理记录时高亮显示，提醒管理员及时处理
26. **通知铃铛高亮**：有新消息时通知铃铛显示红色渐变背景和脉冲动画，提升用户体验
27. **长期租赁状态动态判断**：根据当前日期和开始/结束日期自动判断状态（已支付待使用、使用中、已完成）
28. **退款时间限制**：预约30分钟退款窗口，长期租赁2小时退款窗口，超过时间无法退款
29. **会员订单自动取消**：5分钟未支付自动取消，保留订单记录，状态更新为已取消

## 已知限制

1. 支付功能使用支付宝沙箱环境，生产环境需要配置正式环境

4. 部分功能需要进一步优化性能（如大量数据查询）
